# Niche Jewelry Store (CLI)

A simple, complete implementation of the original Java skeleton using:
- **MVC** (Model: domain classes + StoreModel, Controller: StoreController, View: ConsoleView)
- **Factory Pattern** (ProductFactory, PaymentProcessorFactory)

## Run
```bash
javac *.java
java App
```

## Features
- Browse products
- Cart (add/update/remove/clear)
- Checkout (shipping + payment)
- Order creation + stock decrement
- Admin panel (create/update stock/delete, view orders)
